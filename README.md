# ŠPD Unsko Sanske Šume — Primka Sortimenata

Terenski premjer i primka drvnih sortimenata. PWA aplikacija — radi offline.

## 📱 Kako instalirati na telefon

1. Otvori link aplikacije u Chrome-u (Android) ili Safari-u (iPhone)
2. Klikni **"⬇ Instaliraj App"** dugme — ili u browseru: `⋮ → Dodaj na početni ekran`
3. Aplikacija se pojavljuje kao ikona, radi potpuno offline

## 🚀 Postavljanje na GitHub Pages (besplatno hosting)

### Korak 1 — GitHub nalog
- Idi na [github.com](https://github.com) i napravi besplatan nalog

### Korak 2 — Novi repozitorij
- Klikni **"New repository"**
- Naziv: `spd-primka` (ili bilo šta)
- Označi **Public**
- Klikni **"Create repository"**

### Korak 3 — Upload fajlova
Klikni **"uploading an existing file"** i dodaj SVE ove fajlove:
- `index.html`
- `manifest.json`
- `sw.js`
- `icon-192.png`
- `icon-512.png`

### Korak 4 — Uključi GitHub Pages
- Idi u **Settings → Pages**
- Source: **Deploy from a branch**
- Branch: **main**, folder: **/ (root)**
- Klikni **Save**

### Korak 5 — Link je živ! 🎉
Za ~2 minute dobijate link oblika:
```
https://TVOJE_IME.github.io/spd-primka/
```
Pošalji link svim korisnicima!
